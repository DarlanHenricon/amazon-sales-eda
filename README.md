# 🛒 Análise Exploratória — Vendas Amazon

Exploração de dados de vendas da Amazon investigando
distribuição de preços, volume por categoria e receita por região.

## Ferramentas
Python | pandas | matplotlib | seaborn

## O que foi analisado
- Distribuição e faixas de preço
- Receita e volume por categoria de produto
- Comportamento de compra por região
- Correlação entre preço e quantidade vendida
- Heatmap categoria × região

## Como executar
pip install pandas matplotlib seaborn
jupyter notebook Amazon_Analise_Portfolio.ipynb
